# Diccionario de Traducción: Vizzuality → Entrelampistas

Cada vez que diseñes una escena de un ensayo digital, consulta esta tabla.
La columna izquierda es lo que Vizzuality hace. La columna derecha es cómo
Entrelampistas lo traduce a su mundo.

---

## APERTURAS

| Vizzuality | Entrelampistas |
|---|---|
| Globo 3D interactivo que el usuario gira | Frase provocadora en pantalla completa que el usuario procesa antes de scrollear |
| Dato impactante aislado ("3.6 mil millones de personas") | Dato cotidiano aislado ("4 horas 37 minutos al día dentro de plataformas que no elegiste") |
| Imagen satelital a pantalla completa | Ilustración editorial Chris Ware a pantalla completa |
| Animación de un ecosistema que se transforma | Diagrama que se revela progresivamente mostrando un sistema que te afecta |
| Video corto de impacto | Secuencia de 3-4 frases que aparecen una por una al scroll (como subtítulos de cine) |

**Principio:** Vizzuality abre con algo que el ojo no puede ignorar. Entrelampistas abre con algo que la mente no puede ignorar. Ellos usan la escala del planeta. Tú usas la escala del pensamiento individual.

---

## DATOS Y EVIDENCIA

| Vizzuality | Entrelampistas |
|---|---|
| Mapa de deforestación que cambia al scrollear (antes/después) | Diagrama de cómo se forma una decisión: primero ves el resultado, después ves las capas de influencia que se revelan al scroll |
| Gráfico de emisiones de CO2 por país | Línea de tiempo visual de cómo una narrativa cultural se instala: primer contacto → repetición → normalización → invisibilidad |
| Mapa de calor de vulnerabilidad climática | Mapa visual de los puntos de tu día donde un algoritmo decide por ti (mañana: feed de noticias, mediodía: rutas de Google Maps, noche: recomendaciones de Netflix) |
| Counter animado: "1,000 hectáreas perdidas hoy" | Counter contextualizado: "En el tiempo que llevas leyendo este ensayo, has recibido 47 notificaciones diseñadas para interrumpir tu atención" |
| Infografía de cadena de suministro (soja → deforestación → hamburguesa) | Infografía de cadena de influencia (algoritmo → selección de contenido → opinión formada → decisión tomada) |
| Comparativa visual entre países ricos y países vulnerables | Comparativa visual entre lo que crees que decides libremente y lo que está pre-estructurado por diseño |

**Principio:** Vizzuality hace visible lo que pasa en el planeta y nadie ve. Entrelampistas hace visible lo que pasa en tu cabeza y nadie ve. La técnica es la misma: revelar lo invisible.

## COLOR COMO LENGUAJE

| Vizzuality | Entrelampistas |
|---|---|
| Verde = bosque intacto, naturaleza | var(--paper) = estado de calma, reflexión, espacio para pensar |
| Rojo = pérdida, alerta, deforestación | var(--rojo) = punto de atención, dato que importa, frase central |
| Azul = agua, océano, datos | var(--klein-deep) = estructura profunda, sistema, lo que sostiene |
| Amarillo = zona de riesgo | var(--paper-warm) = pausa reflexiva, cambio de contexto, calidez |
| Gris = datos neutros, contexto | var(--ink-light) = metadata, contexto secundario, lo que acompaña |
| Fondo oscuro = momento de impacto dramático | var(--klein-deep) como fondo = la escena central del ensayo, el giro |
| Cambio de paleta entre secciones = cambio de tema | Cambio de background entre escenas = cambio de acto narrativo |

**Principio:** En Vizzuality el color codifica datos ambientales. En Entrelampistas el color codifica estados cognitivos: calma (paper), atención (rojo), estructura (klein), pausa (warm), profundidad (klein-deep). Nunca uses un color sin razón.

## INTERACCIÓN COMO COMPRENSIÓN

| Vizzuality | Entrelampistas |
|---|---|
| Globo 3D que giras para explorar biodiversidad | Sección expandible donde el lector elige qué rama de un argumento explorar primero |
| Quiz: "¿Sabes cuántas especies están en peligro?" | Pregunta reflexiva interactiva: "¿Cuántas decisiones tomaste hoy que crees que fueron tuyas?" (con respuesta que se revela al click) |
| Mapa donde haces click en un país y ves sus datos | Diagrama donde haces click en un sistema (tecnología, narrativa, estructura social) y ves cómo te afecta |
| Slider temporal: arrastra para ver deforestación 2000→2024 | Slider conceptual: arrastra para ver cómo una creencia se transforma de "sentido común" a "estructura diseñada" |
| Dashboard con filtros por tema/región | Índice expandible del ensayo donde eliges por qué ángulo entrar al tema |
| Tooltip al hover: dato contextual sobre un punto del mapa | Tooltip al hover: definición o contexto de un concepto mencionado en el ensayo |

**Principio:** En Vizzuality la interacción hace que entiendas datos que no entenderías leyendo una tabla. En Entrelampistas la interacción hace que el lector se detenga y piense algo que no pensaría solo leyendo un párrafo. La interacción no es entretenimiento — es herramienta de pensamiento.

---

## SCROLL COMO NARRATIVA

| Vizzuality | Entrelampistas |
|---|---|
| Scroll revela capas de datos sobre un mapa (deforestación → emisiones → impacto social) | Scroll revela capas de un argumento (observación concreta → sistema subyacente → implicación personal) |
| Parallax: la imagen satelital se mueve más lento que el texto | Parallax sutil: la ilustración se mueve más lento que el texto (max 30px) |
| Escena sticky: un mapa se queda fijo mientras los datos cambian | Escena sticky: la frase central del ensayo se queda fija mientras los argumentos que la sostienen scrollean alrededor |
| Progress bar de lectura sobre un caso de estudio | Progress bar de lectura (2px, var(--rojo), fija en top) |
| Transición entre casos: cambio de fondo + nuevo visual | Transición entre actos: cambio de background (paper → paper-warm → klein-deep → paper) |
| Scroll horizontal para mostrar una secuencia temporal | NUNCA usar scroll horizontal. En Entrelampistas, la secuencia temporal se muestra con scroll vertical + cambio de escena |

**Principio:** En Vizzuality el scroll avanza el mapa y los datos. En Entrelampistas el scroll avanza el pensamiento. El lector no está "bajando la página" — está caminando a través de una idea. Cada scroll es un paso en la reflexión.

---

## ESTRUCTURA NARRATIVA

| Vizzuality (4 actos de un caso) | Entrelampistas (4 actos de un ensayo) |
|---|---|
| **ACTO 1:** Imagen satelital impactante — "mira lo que pasa" | **ACTO 1:** Situación cotidiana reconocible — "esto te pasa a ti" |
| **ACTO 2:** Datos que cuantifican el problema — mapas, gráficos, cifras | **ACTO 2:** Evidencia de los sistemas que operan debajo — datos, ejemplos, conexiones que el lector no veía |
| **ACTO 3:** Perspectiva que reencuadra — "esto significa que..." | **ACTO 3:** La idea que cambia cómo ves lo que te rodea — la frase central, el giro cognitivo |
| **ACTO 4:** Herramientas para actuar — "esto puedes hacer" | **ACTO 4:** Resonancia — una imagen o pregunta que el lector se lleva pensando. No instruye — acompaña. |

**Diferencia clave:** Vizzuality cierra con acción (actúa, dona, monitorea). Entrelampistas cierra con resonancia (piensa, observa, quédate con esto). Entrelampistas no le dice al lector qué hacer — le da algo con qué pensar.

## TIPOGRAFÍA COMO ARQUITECTURA

| Vizzuality | Entrelampistas |
|---|---|
| Tipografía grande para datos numéricos | Tipografía grande (1.8-2.5rem) para frases provocadoras y datos |
| Sans-serif limpia para cuerpo de texto | Neo Grotesque SC para todo. Siempre. Sin excepciones. |
| Labels en mayúsculas para categorías (CLIMATE, NATURE, PEOPLE) | Section labels en mayúsculas (RELACIÓN CON LA TECNOLOGÍA, ENTRENAMIENTO COGNITIVO) |
| Texto minúsculo para créditos y fuentes | Texto minúsculo (0.65-0.7rem) para metadata: fecha, categoría, tiempo de lectura |
| Bold para énfasis dentro de párrafos | NO usar bold para énfasis. Usar color var(--rojo) o var(--klein-vibrant) |
| Italic para citas o términos técnicos | NUNCA italic. Usar comillas o background var(--rojo-bg) como highlight inline |

## EJEMPLO COMPLETO: UNA ESCENA TRADUCIDA

### Cómo lo haría Vizzuality (clima)
```
[ESCENA: Pantalla completa]
Mapa de calor a pantalla completa mostrando 
temperaturas extremas en el Sahel.
El mapa se colorea progresivamente mientras scrolleas.

[Dato grande centrado sobre el mapa:]
"52°C"
"Temperatura máxima registrada en el Sahel en 2024"

[Al seguir scrolleando, el mapa se reduce 
y aparece texto:]
"Las comunidades agrícolas de esta región dependen 
de lluvias que cada año son más impredecibles..."
```

### Cómo lo hace Entrelampistas (pensamiento)
```
[ESCENA: Pantalla completa, fondo paper-warm]
Ilustración editorial: persona sentada frente a un 
teléfono, pero el teléfono es un edificio con puertas 
y ventanas (estilo Chris Ware).

[Dato grande centrado debajo de la ilustración:]
"4 horas 37 minutos"
font-size: 2.5rem, color: var(--rojo)

"Tiempo promedio diario dentro de plataformas 
diseñadas para retener tu atención."
font-size: 0.85rem, color: var(--ink-mid)

[Al seguir scrolleando, la ilustración se reduce 
con parallax sutil y aparece texto:]
"No elegimos estas infraestructuras. 
Nacimos dentro de ellas..."
font-size: 1rem, line-height: 1.8, clase .reveal
```

## CÓMO USAR ESTA TABLA

1. Cuando diseñes el guion de dirección de un ensayo, abre esta tabla
2. Para cada escena, busca qué haría Vizzuality con ese tipo de información
3. Tradúcelo a la columna de Entrelampistas
4. Escríbelo en el guion con las especificaciones CSS
5. Pásaselo a Claude Code con el brief de dirección creativa

La tabla no se le pasa a Claude Code directamente — es tu herramienta 
de dirección creativa. Tú la consultas, tomas las decisiones, y le das
a Claude Code instrucciones concretas con valores CSS específicos.
